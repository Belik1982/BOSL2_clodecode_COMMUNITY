FAQ

Do I need to know OpenSCAD to use this? ▼

No. The Customizer panel in OpenSCAD gives you sliders and dropdowns for every parameter. You never need to touch the code. If you can use PowerPoint, you can use this.

Which slicer does it work with? ▼

Any slicer that accepts STL — Bambu Studio, PrusaSlicer, OrcaSlicer, Cura, etc. Export STL from OpenSCAD, import into your slicer. That's it.

How fast does it render? ▼

With the Manifold backend (free, available in OpenSCAD 2024 dev snapshot) most enclosures render in 1–3 seconds. Enable it at Edit → Preferences → Features → manifold.

What do I get after purchase? ▼

Full OpenSCAD source .scad file with all PRO features unlocked, documentation in EN/UA/RU, and all future updates — forever, no subscription.

Can I use it commercially? ▼

Yes. The PRO license covers commercial use — you can use it to design and sell enclosures for your products. The free Community version is non-commercial only.

My folder name has Cyrillic/Chinese characters — will it work? ▼

OpenSCAD requires Latin characters in the full folder path. Place the project in a path like C:\Projects\enclosure — not C:\Проекти\.
