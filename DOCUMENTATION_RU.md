## 1. Установка

### Шаг 1 — Установите OpenSCAD

Скачайте на [openscad.org](https://openscad.org/downloads.html).  
Рекомендуется **dev snapshot** (2024.xx) — включает backend **Manifold**, который рендерит в 10–50× быстрее стандартного CGAL.

После установки dev snapshot:
1. Меню **Edit → Preferences → Features**
2. Включите **"manifold"**
3. Перезапустите OpenSCAD

### Шаг 2 — Откройте проект

1. Запустите OpenSCAD
2. **File → Open** → выберите `enclosure.scad`
3. Нажмите **Ctrl+Shift+C** для открытия панели Customizer
4. Нажмите **F5** для быстрого предпросмотра

### Шаг 3 — Укажите версию

В начале файла `enclosure.scad`, строка 12:
```scad
_EDITION = "community";   // измените на "pro" для PRO версии
```

## 2. Первые шаги

### Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| **F5** | Быстрый предпросмотр (для навигации) |
| **F6** | Полный рендер (для экспорта STL) |
| **F7** | Экспорт файла после F6 |
| **Ctrl+Shift+C** | Открыть / закрыть Customizer |
| **Ctrl+4 / 5** | Вид сверху / изометрия |
| **PageUp / Down** | Масштаб |

### Режимы отображения

Параметр `Part_Chastyna` в группе 01:

| Значение | Отображение |
|----------|-------------|
| `Всі деталі (All)` | База + крышка рядом (для печати) |
| `База (Base)` | Только основание |
| `Кришка (Lid)` | Только крышка |
| `Зібраний (Assembled)` | Корпус в сборке |
| `Напіввідкритий (Ajar)` | Крышка приоткрыта (контроль стыка) |
| `Стійки PCB (PCB Standoffs)` | Стойки отдельно (для отдельной печати) 🔒 |
| `Панель X (X-Panel)` | Съёмная панель левая/правая 🔒 |
| `Панель Y (Y-Panel)` | Съёмная панель передняя/задняя 🔒 |

### Типичный порядок работы

1. Группа **02** — размеры корпуса (L × W × H)
2. Группа **03** — толщина стенок
3. Группа **04** — профиль стыка
4. Группа **05** — тип крепления крышки
5. Группы **07–14** — опциональные функции (панели, PCB, разъёмы, вентиляция...)
6. Группа **15** — текстовые надписи
7. **F5** после каждого изменения для контроля
8. **F6 → F7** для экспорта STL

| Параметр | Значения | По умолч. | Описание |
|----------|----------|-----------|----------|
| `Panel_Enable` | true/false | false | Включить съёмные панели |
| `Panel_Walls` | Перед+Зад / Лево+Право / Все 4 | Перед+Зад | Какие стенки съёмные |
| `Panel_Cl` | 0.1–0.5 мм | 0.2 | Зазор скользящей посадки |

> Высота и ширина шипа рассчитываются автоматически из толщины стенок и дна. Крышка блокирует выход панелей — снять можно только без крышки.

## 10. Рабочий процесс

```
1. НАСТРОЙКА
   Открыть Customizer → проработать группы 02–17 по порядку
   Нажимать F5 после каждой группы для контроля

2. ПРОВЕРКА
   Part = "Зібраний" → осмотреть собранный корпус
   Part = "Напіввідкритий" → осмотреть стык и внутренность
   Проверить BOM в консоли (PRO)

3. РЕНДЕР И ЭКСПОРТ
   F6 → полный рендер (1–60 сек, зависит от сложности)
   File → Export → Export as STL
   Экспортировать основание и крышку отдельно

4. СЛАЙСИНГ (рекомендуется Bambu Studio)
   Основание: плоским дном вниз, без поддержек
   Крышка: плоским верхом вниз, без поддержек
   Периметры: 5–7 · Слоёв верх/низ: 8 · Заполнение: 20–40%

5. ПЕЧАТЬ И СБОРКА
   Установить термозакладки до закрытия (если режим Heatset)
   Запрессовать магниты (натяг Magnet_Press_Fit ≈ -0.10)
   Установить плату, закрыть крышку
```

## 12. Часто задаваемые вопросы

**Q: Поменял параметр — ничего не изменилось.**  
A: Нажмите F5 для обновления. Проверьте, что `Port_Count` ≥ номера редактируемого слота, и что нужная функция включена (`Panel_Enable = true` и т.п.).

**Q: F6 рендерится очень долго (минуты).**  
A: Включите backend **Manifold**: Edit → Preferences → Features → manifold. Ускорение 10–50×. Также используйте `Render_Quality = "Draft"` для промежуточных итераций.

**Q: Стык закрывается неплотно, виден зазор.**  
A: Увеличьте `Clearance_Zazor` (попробуйте 0.20–0.25 мм). Зависит от калибровки первого слоя принтера.

**Q: Как экспортировать только основание или крышку?**  
A: Установите `Part_Chastyna = "База (Base)"` → F6 → F7. Повторите для крышки.

**Q: Как напечатать стойки PCB отдельно?**  
A: `PCB_Fused_To_Base = false`, затем `Part_Chastyna = "Стійки PCB"` → F6 → F7. 🔒

**Q: Разъём не появляется на выбранной грани.**  
A: Проверьте, что разъём не выходит за край грани. Установите `Port_N_Offset_1 = 0` для центрирования, затем настройте смещение.

**Q: Как получить IP54?**  
A: `Gasket_Groove_Enable = true` + `Joint_Styk = "Сходинка (Ledge)"` + установить O-ring. 🔒

**Q: Где найти рекомендуемую длину винтов?**  
A: В консоли OpenSCAD после F5/F6: `INFO: Recommended screw length = XX mm`. 🔒

**Q: Можно ли использовать кириллицу в надписях?**  
A: Да. Шрифт `Liberation Sans:style=Bold` (входит в OpenSCAD) лучше всего поддерживает кириллицу.
