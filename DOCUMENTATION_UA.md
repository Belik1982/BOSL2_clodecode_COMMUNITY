## 1. Встановлення

### Крок 1 — Встановіть OpenSCAD

Завантажте на [openscad.org](https://openscad.org/downloads.html).  
Рекомендується **dev snapshot** (2024.xx) — включає backend **Manifold**, який рендерить у 10–50× швидше за стандартний CGAL.

Після встановлення dev snapshot:
1. Меню **Edit → Preferences → Features**
2. Увімкніть **"manifold"**
3. Перезапустіть OpenSCAD

### Крок 2 — Відкрийте проєкт

1. Запустіть OpenSCAD
2. **File → Open** → оберіть `enclosure.scad`
3. Натисніть **Ctrl+Shift+C** для відкриття панелі Customizer
4. Натисніть **F5** для швидкого перегляду

### Крок 3 — Вкажіть версію

На початку файлу `enclosure.scad`, рядок 12:
```scad
_EDITION = "community";   // змініть на "pro" для PRO версії
```

## 2. Перші кроки

### Гарячі клавіші

| Клавіша | Дія |
|---------|-----|
| **F5** | Швидкий перегляд (для навігації) |
| **F6** | Повний рендер (для експорту STL) |
| **F7** | Експорт файлу після F6 |
| **Ctrl+Shift+C** | Відкрити / закрити Customizer |
| **Ctrl+4 / 5** | Вид зверху / ізометрія |
| **PageUp / Down** | Масштаб |

### Режими відображення

Параметр `Part_Chastyna` у групі 01:

| Значення | Відображення |
|----------|-------------|
| `Всі деталі (All)` | База + кришка поруч (для друку) |
| `База (Base)` | Тільки основа |
| `Кришка (Lid)` | Тільки кришка |
| `Зібраний (Assembled)` | Корпус у зборці |
| `Напіввідкритий (Ajar)` | Кришка трохи відкрита (контроль стику) |
| `Стійки PCB (PCB Standoffs)` | Стійки окремо (для окремого друку) 🔒 |
| `Панель X (X-Panel)` | Знімна панель ліво/право 🔒 |
| `Панель Y (Y-Panel)` | Знімна панель перед/зад 🔒 |

### Типовий порядок роботи

1. Група **02** — розміри корпусу (L × W × H)
2. Група **03** — товщина стінок
3. Група **04** — профіль стику
4. Група **05** — тип кріплення кришки
5. Групи **07–14** — додаткові функції (панелі, PCB, розʼєми, вентиляція...)
6. Група **15** — текстові написи
7. **F5** після кожної зміни для контролю
8. **F6 → F7** для експорту STL

```
   Part = "Зібраний" → оглянути зібраний корпус

   F6 → повний рендер (1–60 сек залежно від складності)
   File → Export → Export as STL
   Експортувати основу та кришку окремо

4. НАРІЗКА (рекомендується Bambu Studio)
   Основа: плоским дном вниз, без підтримок
   Кришка: плоским верхом вниз, без підтримок
   Периметрів: 5–7 · Шарів верх/низ: 8 · Заповнення: 20–40%

   Встановити термозакладки до закриття (якщо режим Heatset)
   Запресувати магніти (натяг Magnet_Press_Fit ≈ -0.10)
   Встановити плату, закрити кришку
```

## 12. Часті запитання

**Q: Змінив параметр — нічого не змінилось.**  
A: Натисніть F5 для оновлення. Перевірте, що `Port_Count` ≥ номера редагованого слота, і що потрібна функція увімкнена (`Panel_Enable = true` тощо).

**Q: F6 рендериться дуже довго (хвилини).**  
A: Увімкніть backend **Manifold**: Edit → Preferences → Features → manifold. Прискорення 10–50×. Також використовуйте `Render_Quality = "Draft"` для проміжних ітерацій.

**Q: Стик закривається нещільно, видно зазор.**  
A: Збільшіть `Clearance_Zazor` (спробуйте 0.20–0.25 мм). Залежить від калібрування першого шару принтера.

**Q: Як експортувати тільки основу або кришку?**  
A: Встановіть `Part_Chastyna = "База (Base)"` → F6 → F7. Повторіть для кришки.

**Q: Як надрукувати стійки PCB окремо?**  
A: `PCB_Fused_To_Base = false`, потім `Part_Chastyna = "Стійки PCB"` → F6 → F7. 🔒

**Q: Розʼєм не зʼявляється на вибраній грані.**  
A: Перевірте, що розʼєм не виходить за край грані. Встановіть `Port_N_Offset_1 = 0` для центрування, потім налаштуйте зміщення.

**Q: Як досягти IP54?**  
A: `Gasket_Groove_Enable = true` + `Joint_Styk = "Сходинка (Ledge)"` + встановити O-ring. 🔒

**Q: Де знайти рекомендовану довжину гвинтів?**  
A: У консолі OpenSCAD після F5/F6: `INFO: Recommended screw length = XX mm`. 🔒

**Q: Чи підтримується кирилиця в написах?**  
A: Так. Шрифт `Liberation Sans:style=Bold` (входить в OpenSCAD) найкраще підтримує кирилицю.
